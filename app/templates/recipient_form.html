Note

Les contenus des mails sont automatiquement générés à partir des modèles que vous créerez ici. Pour ce faire, il faut les indiquer dans le Sujet 

subject
et/ou dans le corps du texte 
body
. Ces variables doivent obligatoirement être entourées d'accolades. A titre d'exemple, le corps de texte suivant 
Le salarié {name} {surname}...
, ou 
name
est MANCEL et 
surname
est Benoit donnera ainsi un mail contenant le message suivant 
Le salarié MANCEL Benoit...

    : Nom 
  • surname
    : Prénom 
  • surname
    : Prénom
